## Projects

### 1. Tendon-Driven Prosthetic Hand Prototype
- Designed a tendon-driven prosthetic hand replicating MCP, PIP, and DIP joints
- Applied biomechanical modelling to optimise force transmission and grip mechanics
- Developed validation protocols (grip strength, dexterity, repeatability)
- Iteratively improved performance by analysing friction, tendon routing, and structural limitations

### 2. Proximity-Based Patient Safety Monitoring System
- Built a real-time monitoring system using Arduino and C++
- Implemented SAFE / WARNING / DANGER classification logic
- Applied signal processing (filtering, hysteresis) for stability
- Achieved ~±1 cm accuracy through experimental validation

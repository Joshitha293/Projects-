## Testing & Validation

Quantitative testing protocols were developed to evaluate performance:

### 1. Cylindrical Grip Test
- Object: Standard cylindrical object (bottle)
- Trials: 20
- Successful grasps: 18
- **Success Rate: 90%**

---

### 2. Precision Grip Test
- Object: Small object (small block)
- Trials: 20
- Successful grasps: 8
- **Success Rate: 40%**

---

### 3. Repeatability Test
- Repeated grasp cycles: 10 cycles
- Consistent performance achieved in 8 cycles
- **Repeatability: 80%**

---

### Summary of Results
- High performance in power grips (cylindrical objects)
- Reduced performance in precision tasks due to mechanical limitations
- Overall system demonstrates functional grasping capability with scope for optimisation
